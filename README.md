# Todo API with ASP.NET Core and MongoDB

## Endpoints

```shell
# Get all items
GET /api/todoitems

# Get one bt id
GET /api/todoitems/:id

# Create todo item
POST /api/todoitems

# Update item
PUT /api/todoitems/:id

# Delete item
DELETE /api/todoitems/:id
```

## Running the app

```shell
docker-compose up -d
```
